SLiMS aka SOD (SLiMS on Drupal) module:
------------------------
Requires - Drupal 7
License - GPL (see LICENSE)


Overview:
--------
Modul SOD ini berguna bagi yang ingin mempublikasikan bibliografi
yang terdapat di SLiMS (Senayan Library Management System) menggunakan
antarmuka Drupal.


Features:
---------
SOD module:

* SLiMS server configuration
* Simple search block


Installation:
------------
Install this module in the usual way (drop the file into your site's
contributed modules directory).


Configuration:
-------------
Go to "Configuration" -> "SLiMS" to find
all the configuration options.
